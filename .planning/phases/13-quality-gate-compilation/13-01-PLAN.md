<task type="auto">
  <name>Task 2: GREEN - Implement gate compilation</name>

  <action>
In compiler.ts:

1. Add import: `import { evaluate } from "./expression.js";` and `import type { QualityGateValidator } from "./types.js";`

2. Create internal helper `compileGateValidator(checkExpression: string, message?: string): QualityGateValidator`:
   - Returns a function `(output: unknown) => { passed: boolean; message?: string }`
   - The function calls `evaluate(checkExpression, { output })` -- wrapping the output in context as `output` so expressions like `{{ output.valid == true }}` work
   - If evaluate returns truthy: return `{ passed: true }`
   - If evaluate returns falsy: return `{ passed: false, ...(message ? { message } : {}) }`
   - The checkExpression already contains `{{ }}` delimiters as stored in the spec

3. Update `compileStep` to populate `qualityGates`:
   - Initialize `const gates: QualityGateValidator[] = []`
   - If `step.verification` exists: push `compileGateValidator(step.verification.check, step.verification.on_fail_message)`
   - If `spec.quality_gates?.pre_output` exists: for each gate, push `compileGateValidator(gate.check, gate.message)`
   - Set `qualityGates: gates` in the return object (replacing the hardcoded `[]`)

Run `npx vitest run packages/core/compiler.test.ts` -- ALL tests must pass (GREEN).
