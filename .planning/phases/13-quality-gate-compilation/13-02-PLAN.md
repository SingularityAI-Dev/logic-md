   - If `selfVerification.enabled === false`, return null
   - If strategy is "rubric" and `selfVerification.rubric` exists:
     - Build a prompt string with sections:
       ```

       Review your output against the following criteria and score each from 0.0 to 1.0:
       
       - **{criterion.name}** (weight: {criterion.weight}): {criterion.description}
       - ...
       
       Calculate your weighted total score. Minimum passing score: {minimumScore}.
       ```
     - minimumScore = `selfVerification.rubric.minimum_score ?? 0.5`
     - Return `{ prompt, minimumScore }`
   - If strategy is "reflection" and `selfVerification.reflection` exists:
     - Use the reflection prompt directly: `selfVerification.reflection.prompt ?? "Review your output for accuracy and completeness."`
     - Return `{ prompt, minimumScore: 0 }` (no scoring for pure reflection)
   - For other strategies (checklist, critic) or missing config: return null (future phases can extend)

3. Update `compileStep` to populate `selfReflection`:
   - If `spec.quality_gates?.self_verification` exists: set `selfReflection: compileSelfReflection(spec.quality_gates.self_verification)`
   - Otherwise: keep `selfReflection: null`
